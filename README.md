# SampleRepo

A React JSX-based notification list component with categorization and scalable architecture.

![Notification List Component](https://github.com/user-attachments/assets/9d9e89aa-478c-4590-9133-c5c1b622dcb5)

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Features

- ✅ Dynamic notification rendering from data arrays
- ✅ Category-based grouping (General, Upgrade History, Security, etc.)
- ✅ Type-based styling (info, success, warning, error)
- ✅ Timestamp display with proper formatting
- ✅ Responsive design for all devices
- ✅ Scalable architecture for future enhancements

For detailed documentation, see [NOTIFICATION_COMPONENT.md](./NOTIFICATION_COMPONENT.md).
